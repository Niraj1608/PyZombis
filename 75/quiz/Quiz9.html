A. first_last6
compruebe si 6 es el primer o el último elemento en la lista de números
first_last6([1, 2, 6]) -> True
first_last6([6, 1, 2, 3]) -> True
first_last6([3, 2, 1]) -> False

B. same_first_last
devuelve True si la lista de números tiene al menos un elemento
y el primer elemento es igual al último
same_first_last([1, 2, 3]) -> False
same_first_last([1, 2, 3, 1]) -> True
same_first_last([1, 2, 1]) -> True

C. common_end
Dada duas listas a e b verifica se os dois primeiros são
iguais ou os dois últimos são iguais
suponha que as listas tenham pelo menos um elemento
common_end([1, 2, 3], [7, 3]) -> True
common_end([1, 2, 3], [7, 3, 2]) -> False
common_end([1, 2, 3], [1, 3]) -> True
